Data File
TGCapital saves your trading data to either a local tgcapital-data.json file or Google Drive appDataFolder. Browser localStorage only keeps storage location metadata, not your trade ledger.
① First time: import your Schwab CSV/JSON below, then save to file or connect Google Drive.
② Next time: open your saved file or click ☁ Connect Google Drive to load your synced data.
③ After each new Schwab import: data auto-saves to the connected backend (file handle or Drive).
⚠ Local file save needs Chrome/Edge 86+. Drive sync works on modern browsers with popup auth enabled.
